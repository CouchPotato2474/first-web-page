Elements and Tags

What is an element in HTML? - According to codeacademy, an element is a unit of content in an HTML Document. An element generally consists of a start tag, content, and an end tag.

Example: the < p > This is the content of our paragraph element

What do elements allow us to do? - Elements allow us to create things for our webpages.

What is a tag? - A tag looks like this <> and surrounds an element.

What does an attribute do? - An attribute identifies a property for an element. An attribute appears within an element's start tag.
